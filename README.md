Require extension extension=gd from XAMPP to use a function imagecreatefromjpeg() that used in this web.

To use the extension is from Apache config named php.ini. 
If you already opened the config, you can use Ctrl + F and find ;extension=gd.
Delete the ; and save the config. Save this config after youre done with it.
Restart the Apache in XAMPP.

This web require database named alzipetshop, you can find the sql at this repo. 
Import that database in your system.
